## 🚀 Quick Start | 快速开始

```bash
# Install | 安装
npm install @norejs/prefetch

# Setup Service Worker | 设置 Service Worker
npx prefetch install --dir public

# Use in your app | 在应用中使用
import { setup, createPreRequest } from '@norejs/prefetch'
```
